About the Machine

Firefox Tweet Machine is a visualization of Firefox activity on Twitter designed and put together by Quodis for Mozilla.

It is an experiment with exclusively open web tools and technologies, and works on all modern browsers that support HTML5 and CSS3.

Technologies used: Twitter API's (Search & REST); Box2DJS (ported to jQuery) for the physics; PHP, Simple PHP Proxy, Memcached, YAML loader and Twitter-Async for the backend; Raphäel JS for the gauge SVG animation; jQuery for easier javascript coding; ColorBox, history, Preload CSS Images and timeago plugins; Topsy for the RT functions; Yanone Kaffeesatz loaded through Google Font API for the interface type and The Lobster Font for the logo.

Keyboard shortcuts

p = play/pause the machine; g = invert gravity; s = jump to search field; a = open the "About the Machine" window; esc = deselect search field / close the "About the Machine" window
